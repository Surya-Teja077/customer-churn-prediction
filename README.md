## 🚀 Project Overview
This project predicts customer churn using Machine Learning models.  
The objective is to identify customers likely to discontinue services and enable proactive retention strategies.

---

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- SMOTE
- Streamlit

---

## 🔄 Machine Learning Pipeline
1. Data preprocessing  
2. Handling class imbalance using SMOTE  
3. Feature scaling  
4. Model training (Logistic Regression, Random Forest, XGBoost)  
5. Hyperparameter tuning using GridSearchCV  
6. Model evaluation using ROC-AUC  

---

## 📈 Model Performance

| Model | Accuracy | ROC-AUC |
|--------|----------|----------|
| Logistic Regression | 82% | 0.84 |
| Random Forest | 86% | 0.88 |
| XGBoost | **89%** | **0.91** |

---

## 📊 Business Insights
- Month-to-month contracts show higher churn
- Low tenure customers are high-risk
- High monthly charges increase churn probability
